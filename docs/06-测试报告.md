## 6. 已知边界

- `/ui` 是 ANSI 流式控制台概览；主交互不接管全屏，采用行式控制台 + SSE 流式输出。
- `/search <query>` 使用 Bing RSS 和 DuckDuckGo Instant Answer 做轻量 Web 检索；默认直连，也可通过 `web_search.proxy_url` 配置专用代理，结果质量取决于网络、代理和上游结构化返回。
- 当前支持内置 MCP 工具和配置文件启用的外部 stdio MCP 工具自动注册；尚未支持 resources、prompts、SSE/HTTP transport。
- Anthropic-compatible `/anthropic` 接口尚未接入；当前支持 OpenAI-compatible `/chat/completions`。
- 真实 API 验证依赖用户自己的有效 API Key。

## 7. 结论

v0.1.0 已满足 M0-M8 课程演示版要求：可构建、可运行、可测试、可调用模型、可执行工具、可在写入/编辑前预览 diff 并确认高风险操作、可记录和回放历史，并具备 AGENTS.md 项目规范、Web 搜索、MCP 最小客户端和跨环境稳定性说明。
