## 6. 已知边界

- `/ui` 是 ANSI 流式控制台概览；主交互不接管全屏，仍是同步阻塞的行式控制台。
- `/search` 是 Web 搜索扩展占位，不会真实联网搜索。
- 当前支持内置 MCP 工具和配置文件启用的外部 stdio MCP 工具自动注册；尚未支持 resources、prompts、SSE/HTTP transport。
- Anthropic-compatible `/anthropic` 接口尚未接入；当前支持 OpenAI-compatible `/chat/completions`。
- 真实 API 验证依赖用户自己的有效 API Key。

## 7. 结论

v0.1.0 已满足课程演示版要求：可构建、可运行、可测试、可调用模型、可执行工具、可在写入/编辑前预览 diff 并确认高风险操作、可记录和回放历史，并具备清晰的扩展边界说明。
