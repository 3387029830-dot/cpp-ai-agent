# cpp-ai-agent —— 项目总结

> 版本：v0.1.0  
> 日期：2026-07-06

## 1. 项目概述

`cpp-ai-agent` 是一个使用 C++17 实现的终端 AI 编程智能体。项目通过 OpenAI 兼容 API 连接大语言模型，并通过本地工具完成文件读取、写入、局部编辑和命令执行等任务。

项目面向课程实践和答辩演示，重点展示从需求、设计、编码、测试到交付的完整工程过程。

## 3. 核心设计

项目采用分层设计：

- `core`：消息、角色、会话。
- `llm`：OpenAI-compatible API 请求、`tool_calls` 解析。
- `tools`：统一工具接口、注册表、文件和命令工具。
- `agent`：Agent 主循环，负责任务编排。
- `security`：权限确认和危险命令拦截。
- `storage`：JSONL 日志和历史回放。
- `diagnostics`：配置诊断和交付排错。

## 4. 交付能力

当前版本支持：

- 多轮终端对话。
- `.env`、系统环境变量、`settings.json` 多级配置。
- LinkAPI 和 DeepSeek 等 OpenAI-compatible provider。
- 模型自动调用 `read_file` 等工具。
- 写入、编辑、命令执行前进行权限确认。
- JSONL 会话日志和历史回放。
- FTXUI 状态页。
- `/doctor` 配置诊断。
- `/demo` 演示脚本。
- `/skills`、`/search`、`/mcp-demo` 扩展演示。

## 5. 安全策略

- API Key 不进入仓库。
- `.env` 被 `.gitignore` 忽略。
- 文件路径限制在 workspace 内。
- 工具风险分级为 Safe、Write、Dangerous。
- Write/Dangerous 工具根据配置进行确认。
- 明显危险命令会被拦截。
