```text
PS D:\cpp-ai-agent> Agent 主流程
用户问题
└─ 进入 Session
   └─ ContextManager 构建上下文窗口
      ├─ 保留 system prompt
      ├─ 保留最新消息
      └─ 控制发送给模型的消息数量
         └─ LLM 推理
            ├─ 不需要工具
            │  └─ assistant 直接回答
            └─ 需要工具
               └─ 生成 tool_call
                  ├─ ToolRegistry 查找工具
                  ├─ PermissionManager 判断风险
                  │  ├─ Safe：直接执行
                  │  ├─ Write：展示 diff 并确认
                  │  └─ Dangerous：展示确认界面
                  ├─ 工具执行
                  ├─ 结果写入 tool message
                  └─ 回填给 LLM 继续推理
```
